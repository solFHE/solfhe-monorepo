The security first platform

ZK Compression

sol φ is a groundbreaking project that combines the power of ZK Compression with Solana's high-performance blockchain to create a privacy-preserving, decentralized advertising platform.

Why trust us?

Traditional advertising systems impose adverts that negatively impact the user experience. Users either use platforms for free by watching adverts or pay to block adverts. This creates dissatisfaction for users who see adverts and makes it difficult for advertisers to attract target audiences.

Technical

Data Collection and Processing Module

Collects user behavior data, Performs data processing on local devices using Federated Learning techniques, Provides data minimization with edge computing

ZK Compression Engine

It uses zk-SNARKs (Zero-Knowledge Succinct Non Interactive Argument of Knowledge) protocol. It compresses and anonymizes user data. It reduces the size while maintaining data integrity.

Ad Matching Algorithm

Uses machine learning models trained with Federated Learning. Applies Differentially Private Data Analysis Techniques. Uses Multilayer Perceptron (MLP) Neural Network Architecture

Solana Management

Solana Blinks Integration

Uses Solana Program Library (SPL) token standards. Smart contracts for Blinks are developed in Rust. NFT metadata management is provided with the Metaplex protocol

Federated Learning

Local model training on user devices. Combining model parameters with Secure Aggregation protocol. Differential Privacy-SGD (DP-SGD) algorithm is applied

Interaction Mechanisms

Click-through actions. View-to-Earn model. Gamification elements (rewards, badges)

SOL Token Reward System

Proof of Engagement concept. Dynamic token value determination with bonding curve. Long-term participation incentive with staking mechanism

Smart Contracts

Solana programs developed with Anchor framework. Modular structure with Cross-Program Invocation (CPI). Use of Program Derived Addresses (PDA)

Reward Distribution Mechanism

Custom Solana program for batched transfers. Sybil-resistant authentication system. Transactions requiring multiple signatures with Threshold Signatures

Data Analysis

Big data processing with Apache Spark. Interactive data analysis with Jupyter Notebooks. Real-time dashboards with Grafana

Secure Enclaves

Intel SGX or ARM TrustZone technologies are used. Sensitive operations are performed in an isolated environment. Secure environment is verified with Remote Attestation

Machine Learning Pipeline

ML workflow automation with Kubeflow. A/B testing framework. Automated machine learning (AutoML) techniques
